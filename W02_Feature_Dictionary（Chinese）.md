# Aido Rover 传感器数据特征字典 (W02_Feature_Dictionary)

| 特征编号 | 特征名称 (Name) | 数学公式 (Formula) | 物理逻辑 (Physical Rationale) | 对FAULT预测的预期影响方向 (Direction of Effect) |
| :--- | :--- | :--- | :--- | :--- |
| **1** | **Recency (故障复现间隔)** | $T_{current} - T_{last\_FAULT}$ | 评估设备自上次故障以来的累积运行时间，时间越短代表系统越不稳定。 | 负相关 (值越小，故障概率越高) |
| **2** | **Frequency (警报频次)** | $\sum_{t=current-7d}^{current} \mathbb{I}(Mode_t == ALERT)$ | 反映过去7天内设备触发异常警告的频率，频次高说明潜在隐患多。 | 正相关 (值越大，故障概率越高) |
| **3** | **Monetary (周能耗积分)** | $\int_{week} -\Delta SoC \,dt$ | 计算单机每周累积消耗的电能（SoC下降积分），耗电异常增大通常意味着机械摩擦或电气短路。 | 正相关 (值越大，故障概率越高) |
| **4** | **Motor_Current_RollMean_5** | $\frac{1}{5}\sum_{i=0}^{4} Current_{t-i}$ | 5窗口电机电流滚动均值。反映近期平均负载，均值持续偏高代表电机过载。 | 正相关 (值越大，故障概率越高) |
| **5** | **Motor_Current_RollStd_5** | $\sqrt{\frac{1}{5}\sum_{i=0}^{4} (Current_{t-i} - \mu)^2}$ | 5窗口电机电流滚动标准差。反映电流波动剧烈程度，异常抖动通常由机械卡顿引起。 | 正相关 (值越大，故障概率越高) |
| **6** | **Battery_SoC_RollMean_5** | $\frac{1}{5}\sum_{i=0}^{4} SoC_{t-i}$ | 5窗口剩余电量滚动均值。处于低电量水平时，系统供电不稳定风险增加。 | 负相关 (值越小，故障概率越高) |
| **7** | **Battery_SoC_RollStd_5** | $\sqrt{\frac{1}{5}\sum_{i=0}^{4} (SoC_{t-i} - \mu)^2}$ | 5窗口电量变化率。若短时间内SoC标准差激增，说明电量虚标或存在瞬间大电流放电。 | 正相关 (值越大，故障概率越高) |
| **8** | **IMU_X_RollStd_5** | $\sqrt{\frac{1}{5}\sum_{i=0}^{4} (IMU_{X, t-i} - \mu)^2}$ | 5窗口IMU X轴（前后）滚动标准差。偏高代表车辆在前进方向发生异常剧烈震动。 | 正相关 (值越大，故障概率越高) |
| **9** | **IMU_Y_RollStd_5** | $\sqrt{\frac{1}{5}\sum_{i=0}^{4} (IMU_{Y, t-i} - \mu)^2}$ | 5窗口IMU Y轴（左右）滚动标准差。偏高代表车辆左右晃动剧烈，可能遭遇崎岖地形或侧滑。 | 正相关 (值越大，故障概率越高) |
| **10** | **IMU_Z_RollStd_5** | $\sqrt{\frac{1}{5}\sum_{i=0}^{4} (IMU_{Z, t-i} - \mu)^2}$ | 5窗口IMU Z轴（垂直）滚动标准差。偏高代表垂直方向颠簸严重，易导致硬件松动。 | 正相关 (值越大，故障概率越高) |
| **11** | **RSSI_RollMean_5** | $\frac{1}{5}\sum_{i=0}^{4} RSSI_{t-i}$ | 5窗口网络信号强度均值。信号长期过低会导致通信中断，触发控制丢失故障。 | 负相关 (值越小，故障概率越高) |
| **12** | **Cross_Unit_Z_Current** | $\frac{Current_{i,t} - \mu_{fleet,t}}{\sigma_{fleet,t}}$ | 跨车队单机电流Z-score。将当前机器人的电流与整个车队均值对比，识别跑偏的异常单机。 | 正相关 (绝对值越大，故障概率越高) |
| **13** | **IMU_Magnitude (总振动幅度)** | $\sqrt{IMU_{X}^2 + IMU_{Y}^2 + IMU_{Z}^2}$ | 计算三轴总加速度幅值，综合评估机器人整体受到的物理冲击和震动强度。 | 正相关 (值越大，故障概率越高) |
| **14** | **Current_SoC_Ratio (能效比)** | $\frac{Current_{motor, t}}{SoC_t + \epsilon}$ | 电流与剩余电量的比值（$\epsilon$为防零微量）。当电量变低且电流变大时，电池内阻增大，极易触发欠压故障。 | 正相关 (值越大，故障概率越高) |
| **15** | **Proximity_RollMean_5** | $\frac{1}{5}\sum_{i=0}^{4} Proximity_{t-i}$ | 5窗口接近传感器距离滚动均值。数值持续极小代表长期处于障碍物极近距离，碰撞风险高。 | 负相关 (值越小，故障概率越高) |
